我整理出的实用工具：

  • ctrl + win + t可以固定或取消固定所选窗口，使其始终位于所有其他窗口的顶部，哪怕焦点丢失
  • ctrl + win + c可以打开颜色选择器，在任何应用程序中选取颜色并复制可配置文件到剪贴板
  • win + shift + t可以打开文本提取器
  • win + shift + /可以打开快捷键指南
  • 两次ctrl寻找鼠标
  • win + shift + h鼠标荧光笔

zoomlt

  • ctrl + 1 激活缩放：按鼠标滚轮或向上/向下箭头键放大或缩小。 按 Esc 或鼠标右键退出缩放模式。 按 Ctrl + C 捕获缩放视图，按 Ctrl + S 保存它。 按 Ctrl + Shift 在复制或保存之前裁剪。

  • ctrl + 4 激活实时缩放:按 Ctrl + Shift + 4 激活实时绘图，按 Esc 清除批注或退出。 按Ctrl + 向上/向下调整缩放级别。

  • ctrl +２屏幕绘制： 放大时，按鼠标左键切换绘图模式，按鼠标右键退出。 按 Ctrl + Z 撤消，按 E 清除绘图，按空格键使光标居中。 笔控制 按 Ctrl + 鼠标滚轮 或 Ctrl + 向上/向下调整笔宽度。 颜色 按 R(红色)、G(绿色)、B(蓝色)、O(橙色)、Y(黄色)或 P(粉色)切换颜色。 高亮和模糊 按 Shift + 颜色键表示半透明高亮，按 X 表示模糊，按 Shift + X 表示更强的模糊。 形状 按 Shift 表示线，按 Ctrl 表示矩形，按 Tab 表示椭圆形，按 Shift + Ctrl 表示箭头。 屏幕 按 W 或 K 表示白色或黑色素描本。 按 Ctrl + C 复制，按 Ctrl + S 保存，按 Ctrl + Shift 裁剪。 按t进入文本编辑
