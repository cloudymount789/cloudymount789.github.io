本文是我自用总结的常用的 Bash 命令行技巧，旨在帮助新手快速上手终端的使用，离一名帅气的极客更进一步。内容涵盖基础路径跳转与增删改查,系统与网络监控, 高效进阶技巧, 开发工具联动等，适合linux新手入门，大部分同时也可用于windows系统的windows powershell。

Bash命令行编程

文件夹跳转

1. 看当前所在位置

pwd

（Windows是cd，Linux是pwd看路径）

2. 进入文件夹

cd 文件夹名
# 举例
cd Desktop
cd /home/用户名   # 绝对路径

3. 返回上一级

cd ..

4. 直接回自己家目录（主页）

cd ~

5. 回到上一次所在文件夹

cd -

二、查看文件/列表

1. 列出当前所有文件、文件夹

ls          # 简单列表
ls -l       # 详细信息（权限、大小、时间）
ls -a       # 显示隐藏文件（带 . 的文件）
ls -la      # 详细+隐藏，最常用

三、新建 & 删除 & 复制（谨慎删！）

1. 建文件夹

mkdir 文件夹名
mkdir test

2. 建空文件

touch 文件名.txt
touch note.md
vim 文件名

3. 复制

cp 源文件 目标
cp note.txt /home/备份/
cp -r 文件夹A 文件夹B   # 复制整个文件夹

4. 移动/重命名

mv 旧名字 新名字       # 重命名
mv 文件 /目标路径/     # 移动

5. 删除（⚠️ Linux删了很难找回！）

rm 文件名.txt          # 删文件
rm -r 文件夹名         # 删文件夹+里面所有
rm -rf 文件夹名        # 强制彻底删除（慎用！）

四、查看文件内容

cat 文件名.txt        # 直接全打印
less 文件名.txt       # 分页看（上下翻）
head 文件名.txt       # 看开头几行
tail 文件名.txt       # 看结尾几行（看日志神器）
tail -f 日志.log      # 实时刷新日志

五、系统 & 网络常用

1. 看IP

ifconfig
# 新版也用
ip a

2. 测试网络通不通

ping www.baidu.com

3. 看当前正在运行的程序

ps aux
top        # 实时看进程、占用

4. 清屏

clear

六、解压/压缩

# 解压 zip
unzip 压缩包.zip

# 解压 tar.gz（Linux标配）
tar -zxvf 压缩包.tar.gz

# 压缩成 tar.gz
tar -zcvf 打包名.tar.gz 要打包的文件夹

七、实用快捷键

  • Tab：自动补全文件名/命令，救命神器

  • ↑ ↓：调出之前输过的命令

  • Ctrl + L：快速清屏（等于clear）

  • Ctrl + C：强制停止当前运行的程序

  • vim ... 启动vim编辑器

  • code .在当前工作区打开vscode

  • ·Ctrl + Shift + E：聚焦到左侧文件资源管理器（用上下键选文件）。

  • Ctrl + P：快速搜索文件（输入文件名直接跳转，不需要去文件夹里翻）。

  • 任意位置win + x + i -> 唤起终端(我已配置默认目录为D盘)

这份 Bash 命令行笔记已经非常扎实，涵盖了最常用的基础操作。为了让它更完整、更符合“从入门到进阶”的逻辑，我为你补充了文件搜索、权限管理、管道与重定向这三个核心板块，并提供了一个适合博客首页的 Summary。

八、查找与搜索

# 1. 在当前目录及子目录下查找文件名
find . -name "test.txt"

# 2. 在文件中搜索特定文本内容
grep "error" server.log        # 在 log 中找包含 error 的行
grep -i "error" server.log     # 忽略大小写
grep -r "function" ./src       # 在文件夹内递归搜索

# 3. 查找命令的可执行文件路径
which python

九、权限管理（遇到 Permission Denied 时看这里）

# 1. 修改文件权限
chmod 755 script.sh      # 赋予执行权限
chmod +x script.sh       # 简单直接地赋予执行权限

# 2. 临时提升为管理员权限执行
sudo apt update          # 以 root 权限运行

# 3. 修改文件所属用户
chown user:group file.txt

十、管道与重定向（将命令组合起来）

# 1. 重定向：将结果保存到文件
ls > files.txt           # 覆盖写入
ls >> files.txt          # 追加写入

# 2. 管道符 | ：把前一个命令的结果交给下一个
ps aux | grep "python"   # 在所有进程中过滤出包含 python 的行
cat log.txt | wc -l      # 统计 log 文件有多少行
